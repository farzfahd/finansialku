# finansialku
Finansialku adalah aplikasi Command-Line Interface berbasis Python yang membantu Anda mengelola keuangan pribadi dengan mudah. Aplikasi ini menyediakan fitur pencatatan pemasukan/pengeluaran, pengaturan anggaran harian/bulanan, kategorisasi transaksi, dan laporan keuangan komprehensif dengan penyimpanan data lokal.
